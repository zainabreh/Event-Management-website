# Event Management Website

A responsive **Event Management Website** developed as my **first project at Web3gle** during my internship.  
This project focuses on building structured layouts and responsive UI using modern CSS techniques.

It helped me strengthen my fundamentals in frontend development, especially **CSS Grid layout**.

## Features

- Responsive event listing layout  
- Clean and structured UI design  
- Grid-based layout system  
- Mobile-friendly design  
- Simple and user-focused interface
